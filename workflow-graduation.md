# Workflow Graduation Rubric

## POV → POC
- Workflow clearly defined
- Owner assigned
- Baseline metrics captured
- Economic hypothesis stated

## POC → Pilot
- Prototype validated with real users
- Measurable impact demonstrated
- Governance risks identified
- Adoption signals present

## Pilot → Program
- ROI validated in production
- Stable adoption
- SOPs documented
- Client ownership established

Workflows that fail graduation criteria are paused or retired.
